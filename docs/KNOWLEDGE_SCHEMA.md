# Astrology Knowledge Schema

## Purpose

Defines the canonical structure for all astrology knowledge files.

---

## planets.json

Each planet entry contains:

- id
- english_name
- sanskrit_name
- weekday
- gender
- nature
- caste
- direction
- element
- deity
- color
- gemstone
- metal
- body_parts
- diseases
- professions
- karakatwas
- owns_signs
- exaltation
- debilitation
- moolatrikona
- vimshottari_dasha
- aspects

---

## signs.json

Each sign contains:

- number
- name
- ruler
- element
- modality
- gender
- symbol
- body_part

---

## natural_relationships.json

Contains the permanent planetary relationships.

---

## dignities.json

Contains:

- exaltation
- debilitation
- own_sign
- moolatrikona

---

## houses.json

Contains metadata for houses 1–12.

---

## nakshatras.json

Contains metadata for all 27 nakshatras.
